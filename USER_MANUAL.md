# Claude Usage Tracker — คู่มือใช้งาน

แอพเมนูบาร์ macOS ที่โชว์ **usage ของ Claude** (limit 5 ชั่วโมง + รายสัปดาห์) แบบเรียลไทม์
มี Clawd เดินน่ารักๆ + วงแหวนบอกระดับการใช้งาน

---

## 1. ต้องมีอะไรบ้าง

- **macOS 14 (Sonoma)** ขึ้นไป
- **Claude subscription** (Pro หรือ Max)
- **Claude Code** ติดตั้งในเครื่อง (ใช้สร้าง token) — ถ้ายังไม่มี ติดตั้งจาก https://claude.com/product/claude-code

---

## 2. ติดตั้งแอพ

1. แตกไฟล์ zip
2. ลาก **Claude Usage Tracker.app** ไปไว้ในโฟลเดอร์ **Applications**
3. **เปิดครั้งแรก** — คลิกขวาที่แอพ → เลือก **Open** → กด **Open** อีกครั้งเพื่อยืนยัน
   > แอพนี้ไม่ได้เซ็นผ่าน Apple (แจกกันเองในเพื่อน) ครั้งแรก macOS จะเตือนว่า "unidentified developer" — เปิดด้วยคลิกขวา → Open ครั้งเดียวพอ ครั้งต่อไปดับเบิลคลิกได้เลย
   >
   > ถ้ายังไม่ให้เปิด: ไปที่ **System Settings → Privacy & Security** เลื่อนลงจะเจอชื่อแอพ กด **Open Anyway**
4. จะเห็นไอคอน **Clawd** (ตัวสีส้ม) โผล่บนเมนูบาร์ (มุมขวาบนของจอ)

---

## 3. เชื่อมต่อ (ทำครั้งเดียว)

1. เปิด **Terminal** พิมพ์คำสั่งนี้แล้ว Enter:
   ```
   claude setup-token
   ```
2. เบราว์เซอร์จะเด้งให้ล็อกอิน Claude → อนุมัติ → จะได้ **token** (ขึ้นต้นด้วย `sk-ant-oat01-...`) copy มา
3. คลิกไอคอน Clawd บนเมนูบาร์ → **Preferences…**
4. วาง token ลงในช่อง **Token** → กด **Save & test**
5. ถ้าขึ้น **"Connected — 5-hour limit X%, Weekly limit Y%"** = สำเร็จ 🎉

> **token เป็นของแต่ละคน** ผูกกับบัญชีคุณ — อย่าเอา token คนอื่นมาใช้ ทุกคนต้องสร้างของตัวเอง

---

## 4. อ่านค่ายังไง

**บนเมนูบาร์:** 🦀 Clawd + วงแหวน + ตัวเลข %
- **Clawd เดินเร็วขึ้น** = กำลังใช้งานหนัก, เดินช้า = ว่าง
- **วงแหวน** = ระดับการใช้ 5 ชั่วโมง → **สีส้ม** ปกติ, **สีแดง** เมื่อเกิน 80%
- **ตัวเลข %** = การใช้งานในหน้าต่าง 5 ชั่วโมง

**คลิกไอคอนเพื่อดูรายละเอียด:**
- **5-hour limit** — % + เวลาที่จะรีเซ็ต
- **Weekly limit** — % + เวลาที่จะรีเซ็ต

---

## 5. Preferences

- **Refresh every** — รีเฟรชทุกกี่วินาที (ค่าเริ่มต้น 60)
- **Launch at login** — เปิดแอพอัตโนมัติตอนเปิดเครื่อง

---

## 6. ความเป็นส่วนตัว / ปลอดภัยไหม

- token เก็บไว้**ในเครื่องคุณเท่านั้น** (ไฟล์สิทธิ์ `0600`) — **ไม่แตะ Keychain**
- แอพเรียกแค่ **API ของ Anthropic เอง** ด้วย token ของคุณ ไม่มี server อื่น ไม่มี tracking
- แต่ละรอบรีเฟรชใช้ประมาณ **1 Haiku token** (แทบไม่มีผลกับ limit)
- ต้องการลบ token: Preferences → **Clear**

---

## 7. แก้ปัญหา

| อาการ | วิธีแก้ |
| --- | --- |
| **429 Rate limited** | ใช้บ่อยไปชั่วคราว รอสักครู่แล้วกด Refresh now |
| **401 Token rejected** | token หมดอายุ — รัน `claude setup-token` ใหม่ แล้ววางใน Preferences |
| **"Add a token in Preferences"** | ยังไม่ได้ใส่ token — ทำตามข้อ 3 |
| เปิดแอพไม่ได้ | คลิกขวา → Open หรือ System Settings → Privacy & Security → Open Anyway |

> **หมายเหตุ:** แอพโชว์ได้แค่ **5-hour + weekly (รวมทุกโมเดล)** เท่านั้น
> ส่วน limit แยกตามโมเดล (เช่น Fable) ดูได้ที่ https://claude.ai/settings/usage เท่านั้น
